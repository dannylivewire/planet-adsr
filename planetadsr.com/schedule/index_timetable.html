Tiva Timetable
A truly responsive schedule javascript library.

Introduction

Tiva Timetable is a responsive schedule javascript library with clean and modern flat interface. It is very easy to use yet gives plenty of features and styling options. You can use it for any kind of schedule or events calendar.

Features

Tiva Timetable has many good features for your demand

Fully Responsive

Tiva Timetable is fully responsive and mobile friendly. It display fine on mobile devices.

Easy To Setup

It is so easy to use Tiva Timetable. Just include some css and js files, then display the calendar by html shortcode.

Easy To Customize

There are options for you to customize the timetable like show/hide navigation, start on monday or sunday, ...

Timetable Source

You can manage timetables in json file or feed from your database via php file with ajax.

Layouts

There are 3 layouts for your choice: month, week and list.

Usage

It is so easy to setup and use Tiva Timetable. Just include some css and js files (If your website has already jquery and magnific popup, don't need include them again), then display Tiva Timetable by using html shortcode

Include CSS and JS files
<!-- Timetable CSS Files -->
<link rel="stylesheet" href="assets/css/magnific-popup.css">
<link rel="stylesheet" href="assets/css/timetable.css">

<!-- Timetable JS Files -->
<script src="assets/js/jquery.min.js">
<script src="assets/js/jquery.magnific-popup.js">
<script src="assets/js/timetable.js">
Display by using html shortcode:
<div class="tiva-timetable" data-source="json" data-view="week"></div>

<!-- Params: -->
<!-- data-source : source of timetables. There are 2 options : json (feed timetables from json file) and php (feed timetables from your database via php file). Default value : json. -->
<!-- data-view : layout of timetables. There are 3 options : month, week, list. Default value : month. -->

Monthly View

Display all your timetables via month.

Weekly View

Display all your timetables via week. You can use as day schedule (repeat on every week) or as specific date (like monthly view).

List View

Display all your timetables on list. You can use as day schedule (repeat on every week) or as specific date (like monthly view).
